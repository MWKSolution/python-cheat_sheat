## builtins module

This module is not normally accessed explicitly by most applications, but can be useful in modules that provide objects with the same name as a built-in value.  
most modules have the name **\_\_builtins\_\_** made available as part of their globals. The value of **\_\_builtins\_\_** is normally either this module or the value of this moduleâ€™s **\_\_dict\_\_** attribute.
